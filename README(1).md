# 🛡️ Fichaje — Control Horario de Seguridad Aeroportuaria

Sistema de registro y control de pausas para personal de seguridad en entornos aeroportuarios. Permite a los agentes fichar sus pausas de desayuno, descanso y comida, y a los responsables monitorizar la actividad del equipo en tiempo real.

---

## Funcionalidades

**Para el agente:**
- Fichaje rápido de salida y vuelta para cada tipo de pausa (Desayuno, Descanso, Comida)
- Selección de terminal y jornada de trabajo
- Cronómetro en tiempo real de la pausa en curso
- Resumen diario con duración total de cada pausa
- Firma digital para verificación de identidad
- Acceso desde cualquier dispositivo con navegador (móvil, tablet, ordenador)

**Para el administrador:**
- Panel de monitorización en tiempo real con estado de cada agente
- Historial completo de fichajes con filtros por agente y fecha
- Edición y eliminación de registros individuales
- Exportación e importación de datos en formato JSON
- Gestión del equipo de agentes
- Acceso protegido por PIN

## Tecnologías

- React 18
- HTML5 / CSS3
- LocalStorage para persistencia de datos
- Diseño responsive (móvil, tablet y escritorio)
- Compatible con PWA (se puede añadir a la pantalla de inicio del móvil)

## Uso

Accede a la aplicación desde el enlace proporcionado por tu responsable. Selecciona tu nombre, elige la terminal donde te encuentras y ficha tus pausas pulsando los botones correspondientes.
